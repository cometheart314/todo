# TODOアプリ

React + TypeScript + Vite で作成したフル機能のTODOアプリケーションです。

## 機能

- ✅ **タスクの追加・削除・一覧表示** - 基本的なCRUD操作
- ✅ **完了/未完了の切り替え** - チェックボックスでタスクの状態を管理
- ✅ **優先度設定** - 高・中・低の3段階で優先度を設定
- ✅ **フィルタリング** - すべて/未完了/完了でタスクを絞り込み
- ✅ **永続化** - localStorageを使用してブラウザにデータを保存

## 技術スタック

- **React 18** - UIライブラリ
- **TypeScript** - 型安全なコード
- **Vite** - 高速ビルドツール
- **CSS Modules** - スコープ付きスタイリング

## 必要要件

- **Node.js 20.19+** または **22.12+**
- npm または yarn

## セットアップ

### 1. 依存関係のインストール

```bash
npm install
```

### 2. 開発サーバーの起動

```bash
npm run dev
```

ブラウザで `http://localhost:5173` にアクセスしてください。

### 3. プロダクションビルド

```bash
npm run build
```

ビルドされたファイルは `dist/` ディレクトリに出力されます。

### 4. プレビュー

```bash
npm run preview
```

## 使い方

### タスクの追加

1. 上部のフォームにタスク名を入力
2. 優先度を選択（低・中・高）
3. 「追加」ボタンをクリック

### タスクの管理

- **完了/未完了の切り替え**: チェックボックスをクリック
- **タスクの削除**: 「削除」ボタンをクリック
- **フィルタリング**: 「すべて」「未完了」「完了」ボタンで表示を切り替え

### データの永続化

タスクは自動的にブラウザのlocalStorageに保存されます。ページをリロードしてもデータは保持されます。

## 優先度の色分け

- 🔴 **高**: 赤色
- 🟠 **中**: オレンジ色
- 🟢 **低**: 緑色
